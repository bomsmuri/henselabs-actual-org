# Project Lighthouse: 세부 운영 지침 v1

> 목표: 익명 브랜드 채널 포지셔닝 확립 + Phase 0 오디언스 빌딩 기반 구축
> 담당 에이전트: Owned Media (단독 주도)
> 의존성: Intelligence/Discovery 산출물이 콘텐츠 소스 — 리서치 없으면 콘텐츠도 없다
> 주의: 개인 브랜드 계정 금지 / 대표 겸업 이슈 있음

---

## Phase 0 채널 전략

| 채널 | 포맷 | 빈도 | 목적 |
|---|---|---|---|
| 인스타그램 | 카드뉴스 (10컷 내외) | 주 2~3회 | 리서치 인사이트 큐레이션 |
| 스레드 / X | 짧은 인사이트 포스트 | 주 3~5회 | 실시간 반응·발견 공유 |
| 틱톡 / 릴스 | 숏폼 스크립트 | 주 1회 | 알고리즘 트래픽 확보 |
| 뉴스레터 | 주간 리서치 큐레이션 | 주 1회 | 고관여 오디언스 락인 |

### 채널 원칙
- 모든 채널은 동일 익명 브랜드 페르소나 사용 (대표 신분 비노출)
- 콘텐츠 톤: 교육·큐레이션 중심 / 판매 유도 금지
- 모든 콘텐츠는 대표 승인 후 발행

## 작업 구조: L-1 / L-2 / L-3

### L-1: 채널 포지셔닝 수립

**목적**: 어떤 채널에서, 어떤 오디언스에게, 어떤 가치를 전달하는지 정의

**세부 작업**
1. **페르소나 정의**
   - 익명 브랜드명 + 계정 바이오 초안 (3개 옵션 제시)
   - 타깃 오디언스 프로파일 (직업, 관심사, 콘텐츠 소비 패턴)
   - 경쟁 채널 3개 분석 (유사 포지셔닝 계정의 팔로워 수, 인게이지먼트율, 콘텐츠 유형)
2. **포지셔닝 문장**
   - "우리는 [타깃]에게 [핵심 가치]를 [차별화 방식]으로 제공한다" 형식 3개 옵션
3. **채널별 역할 분담**
   - 각 채널의 1차 목적(인지·탐색·락인 중 하나) 지정

**산출물**: `content/positioning_document.md`

**포맷**:
```
[페르소나 옵션 3개] — 각 옵션의 장단점
[타깃 오디언스 프로파일]
[포지셔닝 문장 3개]
[채널별 역할 분담표]
[대표 승인 상태]
```

### L-2: 톤앤매너 가이드 + 첫 4주 콘텐츠 캘린더

**선행 조건**: L-1 대표 승인 완료

**세부 작업**
1. **톤앤매너 가이드**
   - 말투: 격식체/구어체 선택 + 예시 문장 5개
   - 사용 금지 표현 리스트 (과장, 효능 주장, 개인 노출 등)
   - 비주얼 방향: 색상 팔레트, 폰트, 레이아웃 참고 이미지 3개 (무료 레퍼런스 링크)
   - 해시태그 전략: 인스타그램 기본 태그셋 20개
2. **첫 4주 콘텐츠 캘린더**
   - 채널별 발행 일정 (날짜, 채널, 포맷, 주제 1줄)
   - 소스 리서치 문서 경로 명시 (Intelligence/Discovery 산출물 기반)
   - 각 콘텐츠의 목적 (인지/신뢰/팔로우 유도 중 하나)

**산출물**:
- `content/style_guide.md` — 톤앤매너 가이드
- `content/calendar.md` — 4주 콘텐츠 캘린더

**캘린더 포맷**:
```
| 날짜 | 채널 | 포맷 | 주제 | 소스 문서 | 목적 | 상태 |
```

---

### L-3: 첫 콘텐츠 10개 Draft

**선행 조건**: L-2 대표 승인 완료 + Intelligence 섹션 A 이상 산출물 존재

**세부 작업**
- 캘린더 상위 10개 콘텐츠 full draft 작성
- 채널 믹스: 인스타그램 카드뉴스 4개 + 스레드/X 포스트 4개 + 뉴스레터 1개 + 숏폼 스크립트 1개 (기본값; 캘린더 우선)
- 모든 draft는 소스 리서치 문서 경로 필수 첨부

**각 Draft 포맷**:
```
[채널]
[발행 예정일]
[훅 (첫 문장/첫 컷)]
[본문/슬라이드별 텍스트]
[CTA]
[소스 리서치 문서 경로]
[승인 상태: draft / 대표 검토 중 / 승인 / 발행]
```

**산출물 경로**: `content/drafts/YYYYMMDD_채널_주제_vN.md`

## 콘텐츠 포맷별 세부 기준

### 인스타그램 카드뉴스
- 슬라이드 수: 7~10컷
- 1컷: 훅 (질문형 또는 반직관적 사실)
- 2~9컷: 핵심 내용 (컷당 텍스트 50자 이내)
- 마지막 컷: CTA (저장, 팔로우, 링크)
- 텍스트 + 레이아웃 설명 포함 (디자인 제작용)

### 스레드 / X 포스트
- 길이: 200자 이내 (단문) 또는 5개 이내 스레드
- 형식: 인사이트 1줄 + 근거 2~3줄
- 이모지: 사용 여부는 스타일가이드 승인 후 결정

### 숏폼 스크립트 (틱톡/릴스)
- 길이: 30~60초 기준 스크립트
- 구성: 훅(3초) + 본문(20~40초) + 마무리(5초)
- 내레이션 텍스트 + 화면 설명(자막·비주얼 지시) 포함

### 뉴스레터
- 길이: 600~900자 (읽기 시간 3분 이내)
- 구성: 이번 주 핵심 인사이트 1개 + 리서치 요약 3개 + CTA
- 소스: 해당 주 Intelligence/Discovery 산출물 기반
