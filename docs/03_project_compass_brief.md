## 섹션 B: 흥망 패턴 분석

**담당**: Intelligence

### 조사 항목
1. **성공 패턴** — 한국 D2C 브랜드 중 3년 내 흑자 전환 사례 5개 이상
   - 흑자 전환 시점, 전환 트리거(채널·제품·운영 중 무엇이 결정적이었나)
   - 재구매율·LTV 상승 시점과 마케팅 전략 변화의 상관관계
2. **실패 패턴** — 1년 내 철수 또는 적자 지속 사례 5개 이상
   - 현금 소진 타임라인 (런칭~철수)
   - 광고 의존도 vs 자연 트래픽 비율 변화 추이
3. **패턴 교차 분석**
   - 성공 기업이 공통으로 보유한 "필수 조건" vs 실패 기업이 놓친 "결정적 공백"

- [패턴 매트릭스] 성공/실패 기업을 2×2로 분류 (재구매 구조 유무 × 채널 다각화 유무)

## 섹션 C: 규제 리스크 지도

**담당**: Intelligence

### 조사 항목
1. **식약처 규제**
   - 기능성 화장품 표시·광고 기준 (효능 주장 허용 범위)
   - 의약외품 경계 기준 (어떤 주장이 의약외품 분류를 유발하는가)
   - 최근 3년 내 행정처분 사례 (카테고리별)
2. **공정거래위원회 규제**
   - 표시·광고법 위반 유형 (D2C 브랜드 주요 사례)
   - 추천·보증 광고 기준 (인플루언서·리뷰 마케팅 제약)
   - 할인율 표시 기준 (상시 할인 처벌 사례)
3. **전자상거래법·소비자보호법**
   - 청약철회 기준 및 예외 (소모품, 개봉 제품)
   - 정기결제(구독) 관련 규제 동향
4. **카테고리별 리스크 등급표** (상/중/하)
   - 스킨케어 / 건강식품 / 생활용품 / 반려동물용품 / 유아용품 등 10개 카테고리 이상

### 산출물 포맷
표준 Intelligence 리포트 포맷 + 추가 섹션:
- [카테고리별 규제 리스크 등급표] 10개+ 카테고리, 등급 + 핵심 규제 조항 + 처벌 사례

## 섹션 D: 카테고리 기회 스캔 Top 30

**담당**: Discovery

### 조사 항목
1. **해외 선행 신호**
   - 미국·일본·중국 D2C에서 최근 12개월 내 성장 카테고리 (아마존 베스트셀러, 틱톡샵 트렌드 등)
   - 국내 미유입 또는 유입 초기 카테고리 특정
2. **국내 1차 신호**
   - 네이버·쿠팡·올리브영 검색량·매출 상승 카테고리
   - 커뮤니티(디시, 오픈카카오, 네이버카페) 관심 증가 신호
   - 틱톡·인스타그램 숏폼 바이럴 아이템
3. **카테고리 필터링**
   - 탐색 필터 7개 항목 전체 적용
   - 30개 후보 → 필터 통과 개수 명시

- 종합 테이블: 30개 후보를 필터 통과율 순으로 정렬

## 섹션 E: 1인+AI 운영 가능성 검증

**담당**: 전략-재무 (Intelligence·Discovery 산출물 기반)

### 조사 항목
Intelligence·Discovery가 완료된 후 착수.

1. **운영 복잡도 평가** (섹션 B·D 결과 기반)
   - 재고·물류·CS 운영 시 1인이 AI 지원으로 처리 가능한 카테고리 범위 추정
   - OEM 공장 접근성, MOQ(최소 주문량) 수준, 리드타임
2. **단위경제 시뮬레이션** (섹션 A·B 벤치마크 기반)
   - 후보 카테고리별 가정 CAC / LTV / Gross Margin / Payback Period
   - 흑자 전환 시점 추정 (Base 시나리오)
3. **AI 대체 가능 업무 범위**
   - 콘텐츠 제작 / 리서치 / CS 응대 / 광고 운영 중 AI 의존 가능 비율 추정
   - 대표 직접 개입 필수 업무 정의
4. **리스크-기회 매트릭스** (섹션 C 규제 + 섹션 D 기회 교차)
   - 규제 리스크 낮음 × 기회 신호 강함 카테고리 하이라이트

### 산출물 포맷
표준 전략-재무 리포트 포맷:
[TL;DR] / [가정 테이블] / [시나리오별 결과(Best·Base·Worst)] / [핵심 민감도] / [의사결정 옵션] / [리스크 플래그]

추가 섹션:
- [AI 운영 가능성 평가표] 카테고리별 운영 난이도 × AI 대체율
