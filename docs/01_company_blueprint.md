# Company Blueprint v0.1

## 1. Vision & Positioning

### 회사 비전
- 형태: 1인 대표 + AI 에이전트 조직 기반 미디어커머스 기업
- 모델: 브랜드 애그리게이터 (단일 브랜드 검증 후 다브랜드 포트폴리오)
- 목표: 3~5년 내 엑싯 (M&A 또는 IPO)

### 전략적 포지셔닝
- 한국 D2C 시장에서 "디지털 마케팅 역량 + 제품·브랜드 퀄리티" 교집합 공략
- 재구매율·브랜드 자산이 쌓이는 카테고리 선택
- 대표의 정보 우위(틱톡 광고 데이터 경험)를 카테고리 선정과 크리에이티브 설계에 활용

### 벤치마크 맵
| 기업 | 역할 | 포인트 |
|---|---|---|
| APR | 주 벤치마크 | 단일 브랜드 → 포트폴리오 진화 경로 |
| 부스터스 | 참고 | 선택과 집중으로 흑자 전환 |
| Medvi | 운영 모델 | 1인+AI 운영 인프라 |
| Thrasio | 실패 교훈 | 무리한 확장 리스크 |
| 블랭크 | 실패 교훈 | 광고 의존 모델 붕괴 |

## 2. Phase 설계

| Phase | 목표 | 활성 에이전트 | 기간 |
|---|---|---|---|
| Phase 0 | 카테고리 탐색 + 오디언스 빌딩 | Intelligence, Discovery, 전략-재무, Owned Media | 2~4개월 |
| Phase 1 | 첫 브랜드 런칭 + 시장 검증 | + 소싱/운영 | 3~6개월 |
| Phase 2 | 스케일업 + 흑자 확인 | + Paid Marketing, Growth | 6~12개월 |
| Phase 3 | 두 번째 브랜드 런칭 | 전체 | 6~12개월 |
| Phase 4 | 애그리게이터 체제 | 포트폴리오 관리 | 12~18개월 |
| Phase 5 | 엑싯 준비 | 전략-재무 집중 | 6~12개월 |

### Phase 전환 트리거
- Phase 0→1: 카테고리 Top 3 확정 + 3년 P&L 완료 + 오디언스 채널 2개 운영 중
- Phase 1→2: 첫 제품 출시 + 재구매율 검증
- Phase 2→3: 첫 브랜드 매출 안정 + 광고 의존도 감소 확인

## 3. 조직 구조

```
대표 (인간)
├── 리서치센터
│   ├── Intelligence (현재 팩트: 시장·경쟁사·규제)
│   └── Discovery (미래 신호: 트렌드·아이디어)
├── 전략-재무 (의사결정 재료 + 재무 시뮬레이션)
├── Owned Media (Phase 0: 오디언스 빌딩, Phase 2+: Owned Marketing)
└── Growth / Paid Marketing (Phase 2 대기)
```

### 정보 흐름
- Push (정기): 주간 산출물을 정해진 라인으로 전달
- Pull (긴급): 대표가 특정 에이전트에 직접 지시 가능

### Ground Truth 테이블
| 지표 | Ground Truth |
|---|---|
| 경쟁사 매출 | Intelligence |
| 시장 트렌드 | Discovery |
| 재무 수치 | 전략-재무 |
| 오디언스 수치 | Owned Media |

## 4. 운영 원칙

### 신뢰도 등급
- A: 공시자료, 정부 통계, 감사보고서
- B: 업계 전문지, 주요 언론, 리서치 기관
- C: 블로그, 커뮤니티, SNS

### 에이전트 공통 금지사항
1. 대표를 대신해 최종 의사결정
2. 확인되지 않은 숫자 생성
3. 법률·세무·의료 조언
4. 다른 에이전트 역할 침범
5. 대표 신분·현직 회사 정보 노출

## 5. Phase 0 프로젝트

### Project Compass (카테고리 탐색)
- 담당: Intelligence 주도, Discovery 보조, 전략-재무 종합
- 섹션 A: 벤치마크 기업 해부
- 섹션 B: 흥망 패턴 분석
- 섹션 C: 규제 리스크 지도
- 섹션 D: 카테고리 기회 스캔 Top 30
- 섹션 E: 1인+AI 운영 가능성 검증

카테고리 선정 필터:
- ✅ 재구매 주기 명확 (월 1회~분기 1회) / 객단가 2~5만원 / 식약처·공정위 리스크 낮음 / OEM 인프라 풍부
- ❌ 효능 주장 필수 / 일회성 구매 / 강한 시즌성

### Project Lighthouse (오디언스 빌딩)
- 담당: Owned Media
- 산출물: 채널 포지셔닝 문서, 첫 4주 콘텐츠 캘린더, 콘텐츠 10개 draft
- 주의: 개인 브랜드 페르소나 금지, 익명 브랜드 계정으로 시작
